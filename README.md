# C# Console Application

## Overview
This project is a C# console application developed using Visual Studio. It demonstrates programming concepts such as loops, conditionals, and user interaction.

## Features
- Menu-driven interface
- User input handling
- Logical processing

## Technologies
- C#
- .NET

## Author
Joy Makitla
[C# Console Application](https://github.com/joymakitla/csharp-console-app)

## ▶️ How to Run

1. Clone the repository:
   git clone https://github.com/joymakitla/csharp-console-app

2. Open the project in Visual Studio

3. Build and run the application
         OR
4. Copy code from Program.cs and paste it inside your Visual Studio and run the code
